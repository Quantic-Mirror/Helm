Browser Extension

Helm doesn't have a published extension yet — but you can get most of the same benefit two ways:

Option 1 — New Tab override
In Chrome: install New Tab Redirect, set the URL to this file's local path or hosted URL.
In Firefox: install New Tab Override and point it here.

Option 2 — Bookmarklet
Use the ⊕ Bookmarklet button to add any page to Helm with one click from your toolbar.

Option 3 — Self-host
Run python3 -m http.server 8080 in the folder containing this file, then set your new tab URL to http://localhost:8080/bookmark-manager.html.

⚠ Won't work from a local file
Helm is currently opened as a file:// page. Browsers block JavaScript from navigating to local files for security reasons, so the bookmarklet will silently do nothing.

Fix: serve Helm over http://localhost instead. In the folder containing this file, run:
python3 -m http.server 8080 then open http://localhost:8080/bookmark-manager.html and re-grab the bookmarklet from there.
